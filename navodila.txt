RESERVATIONS 
CREATE TABLE reservations (
    id int(11) AUTO_INCREMENT PRIMARY KEY,
    name varchar(100),
    email varchar(100),
    model varchar(100),
    date_from date,
    date_to date,
    status enum('Na čakanju', 'Potrjeno', 'Zavrnjeno') DEFAULT 'Na čakanju'

);

CRUD
Create (Insert) se nahaja v submit_request.php, ker s pomocjo tega podatke iz form pošiljamo v bazo
Update imamo v slučajo ko admin odobri ali zavrne rezervacijo (approve_request.php, ali reject_request.php)
Delete se nahaja v delete_reservation.php in omogoča uporabniku da izbriše svojo rezervacijo
Read (Select) imamo v myreservation.php in s pomočjo tega dobimo vse rezervacije ki jih ima določeni uporabnik
